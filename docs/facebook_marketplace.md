## 1. CHARAKTERYSTYKA PLATFORMY

### 1.1 Specyfika Facebook Marketplace

**Facebook Marketplace** to zintegrowana z ekosystemem Facebooka platforma sprzedażowa C2C, wykorzystująca siłę sieci społecznościowej do transakcji lokalnych. Główną przewagą jest dostęp do ogromnej bazy użytkowników Facebooka oraz możliwość weryfikacji sprzedawców przez ich profile społecznościowe.

| Aspekt | Charakterystyka |
|--------|-----------------|
| **Model biznesu** | C2C, sprzedaż lokalna z opcją wysyłki |
| **Prowizja** | 0 zł (transakcje lokalne), 5% lub min. 2 zł (wysyłka) |
| **Baza użytkowników** | ~25 mln aktywnych użytkowników FB w Polsce |
| **Komunikacja** | Messenger (natychmiastowa, nieformalna) |
| **Weryfikacja** | Profil Facebook (zdjęcie, znajomi, aktywność) |
| **Zasięg** | Domyślnie lokalny (promień km), możliwość rozszerzenia |
| **Integracja** | Grupy sprzedażowe, Allegro Lokalnie |
| **Limit ogłoszeń** | Brak formalnego limitu |

## 2. TYTUŁ OGŁOSZENIA

### 2.1 Parametry techniczne

| Parametr | Wartość | Uwagi |
|----------|---------|-------|
| **Limit znaków** | 99 znaków | Wyświetla się ~50-60 w podglądzie |
| **Widoczność w podglądzie** | ~50-60 znaków | Reszta obcięta w kafelku |
| **Słowa kluczowe** | Na początku | Krytyczne dla wyszukiwania |
| **Formatowanie** | Normalne | Bez CAPS LOCK, bez emotikon |
| **Styl** | Prosty, bezpośredni | Użytkownicy FB preferują nieformalność |
| **Lokalizacja** | Opcjonalna w tytule | System pokazuje automatycznie |

### 2.2 Formuła struktury tytułu

```
[MARKA/TYP] + [MODEL/NAZWA] + [KLUCZOWY PARAMETR] + [STAN opcjonalnie]
```

**Wersja rozszerzona (jeśli miejsce):**
```
[MARKA] + [MODEL] + [PARAMETR 1] + [PARAMETR 2] + [STAN]
```

### 2.3 Struktura tytułu według kategorii

#### Meble
```
[TYP MEBLA] + [CECHA/MATERIAŁ] + [WYMIAR lub OPIS] + [KOLOR opcjonalnie]
```
**Przykłady:**
- `Sofa narożna rozkładana szara 280cm`
- `Stół dębowy rozkładany 6 krzeseł`
- `Szafa IKEA PAX 200x60x236 biała`
- `Łóżko 160x200 z materacem i stelażem`
- `Komoda 8 szuflad drewno lite`

#### Elektronika
```
[MARKA] + [MODEL] + [POJEMNOŚĆ/PARAMETR] + [STAN opcjonalnie]
```
**Przykłady:**
- `iPhone 13 Pro 256GB grafitowy`
- `Samsung Galaxy S22 128GB jak nowy`
- `MacBook Pro 14 M1 Pro 16GB 512SSD`
- `PlayStation 5 Disc 2 pady`
- `iPad Air 5 64GB WiFi srebrny`

#### Odzież
```
[MARKA] + [TYP] + [ROZMIAR] + [STAN opcjonalnie]
```

## 3. TREŚĆ OGŁOSZENIA (OPIS)

### 3.1 Specyfika opisów na Facebook Marketplace

Facebook Marketplace charakteryzuje się **krótszymi, bardziej nieformalnymi opisami** niż inne platformy. Użytkownicy FB przyzwyczajeni są do szybkiego scrollowania i oczekują kluczowych informacji "od razu".

```yaml
charakterystyka_opisow_fb:
  dlugosc:
    optymalna: "100-200 słów"
    minimum: "50 słów"
    maksimum: "300 słów"
    powod: "Użytkownicy FB preferują krótkie treści"
  
  styl:
    ton: "Nieformalny, przyjazny"
    zdania: "Krótkie, konkretne"
    unikaj: "Zbyt formalnego języka"
  
  formatowanie:
    listy: "Wypunktowane (• lub -)"
    akapity: "Krótkie, 2-3 zdania"
    emoji: "Dopuszczalne w umiarkowanej ilości (1-3)"
  
  pierwsze_zdania:
    znaczenie: "Krytyczne - widoczne bez rozwijania"
    zawartosc: "Co to + stan + główna zaleta"
```

### 3.2 Optymalna struktura opisu (5 sekcji)

```
┌─────────────────────────────────────────────────────────────┐
│ 📦 SEKCJA 1: HOOK - PIERWSZE ZDANIE                         │
│    └─ Co to + stan + główna zaleta (widoczne bez klikania)  │
├─────────────────────────────────────────────────────────────┤
│ 📋 SEKCJA 2: SPECYFIKACJA (skrócona)                        │
│    └─ 4-6 najważniejszych parametrów                        │
├─────────────────────────────────────────────────────────────┤
│ ✅ SEKCJA 3: STAN I WADY                                    │
│    └─ Krótki opis stanu + wady (jeśli są)                   │
├─────────────────────────────────────────────────────────────┤
│ 📍 SEKCJA 4: ODBIÓR I KONTAKT                               │
│    └─ Lokalizacja + dostępność + preferencje kontaktu       │
├─────────────────────────────────────────────────────────────┤
│ 💬 SEKCJA 5: DODATKOWE INFO (opcjonalna)                    │
│    └─ Powód sprzedaży + negocjacje + inne                   │
└─────────────────────────────────────────────────────────────┘
```

### 3.3 Parametry opisu

| Parametr | Wartość optymalna | Minimum | Maximum |
|----------|-------------------|---------|---------|
| **Liczba słów** | 100-200 | 50 | 300 |
| **Liczba sekcji** | 5 | 4 | 5 |
| **Punkty specyfikacji** | 4-6 | 3 | 8 |
| **Emoji** | 1-3 | 0 | 5 |
| **Akapity** | 3-4 | 2 | 5 |
